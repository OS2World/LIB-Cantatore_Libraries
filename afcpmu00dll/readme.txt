This library is used on Alessandro's "Batch Editor" program. 

This version of the DLL contains controls created overriding the class WC_STATIC.

The all000 directory contains a draft of the same controls created from scratch.

Monitoring progress derives from that contained in dir 006.

Control color wheel comes from the source included in the base directory colorwheel.
